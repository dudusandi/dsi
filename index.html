Política de Privacidade

1. Informações coletadas

O Aplicativo não coleta, armazena ou compartilha dados pessoais em servidores externos.

Todas as informações inseridas pelo usuário, como:

  • nome do medicamento ou pílula
  • horários e frequência de uso
  • status de tomada (tomado ou agendado)

são armazenadas exclusivamente no próprio dispositivo do usuário.

2. Dados de saúde

As informações relacionadas a medicamentos e lembretes podem ser consideradas dados de saúde. Esses dados:

  • permanecem apenas no dispositivo do usuário
  • não são transmitidos para servidores
  • não são acessados pelo desenvolvedor

O Aplicativo não realiza diagnóstico, tratamento ou aconselhamento médico.

3. Notificações

O Aplicativo utiliza notificações locais para lembrar o usuário de tomar a pílula nos horários configurados. Nenhuma informação é enviada a terceiros para esse funcionamento.

4. Compartilhamento de dados

O Aplicativo não compartilha dados com terceiros, incluindo anunciantes, empresas de análise ou parceiros comerciais.

5. Permissões

O Aplicativo pode solicitar apenas as permissões necessárias para seu funcionamento, como:

  • envio de notificações

Nenhuma permissão é utilizada para coleta de dados pessoais.

6. Armazenamento e segurança

Como os dados ficam apenas no dispositivo, a segurança das informações depende também das medidas de proteção adotadas pelo próprio usuário, como senha, biometria ou bloqueio do aparelho.

7. Crianças e adolescentes

O Aplicativo não é direcionado especificamente a crianças menores de 13 anos e não coleta intencionalmente dados pessoais de menores.

8. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomenda-se que o usuário revise esta página sempre que houver uma nova versão do Aplicativo.
